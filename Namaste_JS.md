1. The Execution Context (0:15 - 0:42)
Definition: Everything in JavaScript happens inside an Execution Context.
Visualization: Think of it as a "big container" or "box" where the entire JavaScript code is executed.
2. Components of Execution Context (0:42 - 2:11)
An execution context consists of two primary components:

Memory Component (Variable Environment):
This is where variables and functions are stored as key-value pairs (e.g., a: 10).
Code Component (Thread of Execution):
This is the environment where the code is executed one line at a time.
3. Core Fundamentals of JavaScript (2:11 - 3:19)
JavaScript is a synchronous single-threaded language:

Single-threaded: JavaScript can execute only one command at a time.
Synchronous: JavaScript executes code in a specific, sequential order. The engine must finish executing the current line before moving to the next.
