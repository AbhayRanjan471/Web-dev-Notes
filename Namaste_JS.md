1. The Execution Context (0:15 - 0:42)
Definition: Everything in JavaScript happens inside an Execution Context. You can visualize it as a container where your code is executed.
2. Components of Execution Context (0:42 - 2:11)
An execution context consists of two primary components:

Memory Component (Variable Environment): Stores variables and functions as key-value pairs (e.g., a: 10).
Code Component (Thread of Execution): The environment where code is executed one line at a time.
3. Core Fundamentals of JavaScript (2:11 - 3:19)
Synchronous Single-Threaded Language:
Single-threaded: Executes only one command at a time.
Synchronous: Executes code in a strict order; the current line must finish before the next one starts.
